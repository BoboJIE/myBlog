Web

  1. web就是一个网页

  2. 静态

    • 对所有人都是一样的

    • 没有数据的处理

    • 直接就是展示数据

  3. 动态

    • 对数据库进行操作
    • 动态的处理数据

web服务器

Apache、Nginx、IIS

  1. ASP
    • 微软开发的
    • 早期国内用的最多
    • VB的脚本
  2. JSP
    • Sun公司开发的
    • 基于Java语言的，也是Java语言编写的
    • 利用Java来实现脚本
  3. PHP
    • 70%公司都在使用
    • 代码非常简单，开发容易
    • 缺点： 不能承载高访问量

Tomcat

  1. Apache基金会的顶级项目，现在时已经是9.0版本了
  2. 优性能的实现JSP和Servlet
  3. 支持访问量小的服务器

HTTP

  1. HTTP全称超文本传输协议，主要用在TCP上
  2. 两个方面：请求，响应
  3. 请求
    • get，显示传输
    • post，隐士传输
    • 请求数据包
      1. 请求行：http-version、method、request-url
      2. 请求头：属性
      3. 数据体
  4. 响应
    • 请求数据包
      1. 响应行：http-version、status-code、reason-phrase
      2. 响应头：属性
      3. 响应体
  5. HTTPS
    • S指的是secure，安全
    • SSL证书
    • 谷歌浏览器，只要是没有实现HTTPS的网页，都会被标注不安全

Maven

  1. 项目管理工具
  2. 从线上之下载jar包到IDE，不需要人工复制粘贴

Servlet

Service Applet

  1. helloServlet

  2. 配置Tomcat环境

  3. 部署

  4. public class HelloServlet extends HttpServlet {
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        resp.getWriter().println("hello");
    }

    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        doGet(req, resp);
    }
}
    • 继承HttpServlet
    • 重写doGet() doPost()方法
    • 即可实现servlet
