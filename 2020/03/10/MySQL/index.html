数据库分类

关系型数据库

  • MySQL、Oracle、DB2
  • 以行和列的形式存储数据

非关系型数据库

  • NoSQL(not only SQL)
  • Redis、MongoDB

DBMS

  • 数据库管理系统
  • 为管理数据库而设计的大型电脑软件管理系统
  • Oracle、Access、MySQL

MySQL安装

  1. 下载并解压包

    https://dev.mysql.com/downloads/file/?id=492455

  2. 配置环境变量

    • 复制bin目录D:\environment\mysql-8.0.18-winx64\bin

    • 复制到Path即可

  3. 配置mysql.init

    • 进入D:\environment\mysql-8.0.19-winx64

    • 创建my.ini

    • 添加

      shell

      注意：skip-grant-tables用于跳过密码，如果进入mysql界面遇到问题，需要把它注释掉

  4. 安装

    • 启动管理员模式下的CMD，目录切换到mysql下的bin目录，然后输入mysqld -install安装

    • 若发生The service already exists! 则输入sc delete mysql杀服务

    • 若未卸载之前的mysql，参考教程

      https://blog.csdn.net/sxingming/article/details/52601250

  1. 初始化

    • 输入

      shell

  2. 再次启动mysql，mysql -u root -p

    注意p后面千万别加空格，加了就代表设置密码为空格

  3. 进入后更改root密码

    • update mysql.user set authentication_string=password('123456') where user='root' and Host='localhost'
